This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).This is the code base for the Basic Basic Calculator with 4-point precision. Our main target here is to understand the various components and it's interaction with each other.


## Cloning the project
You may clone the project using the command
```
git clone https://github.com/suhpin95/TodoList_Redux_react.git
```

### `npm start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.
